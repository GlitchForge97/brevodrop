# BRÉVOLINE — Inventory

A modern, responsive website for BRÉVOLINE, a premium digital solutions and inventory management service provider.

## Overview

BRÉVOLINE is a professional services website showcasing digital design solutions, portfolio work, and service offerings. The site features modern animations, responsive design, and an intuitive user experience built with Tailwind CSS and vanilla JavaScript.

## Features

- Responsive design optimized for desktop, tablet, and mobile devices
- Smooth scroll-based animations with card reveal effects
- Interactive image gallery with blob-shaped images and carousel controls
- Service showcase with detailed descriptions
- Portfolio section featuring completed projects
- Client testimonials
- Process workflow explanation
- Modern UI with custom color scheme and typography
- Accessibility-focused HTML structure
- Legal pages (Terms of Service and Privacy Policy)

## Setup Instructions

1. Extract the project files to your desired directory
2. Open index.html in a modern web browser
3. No build process or dependencies required

For local development with a live server:
- Use VS Code Live Server extension
- Or use Python: `python -m http.server 8000`
- Or use Node.js: `npx http-server`
